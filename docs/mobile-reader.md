# cark 随身论文阅读器

移动端直接复用 cark 的论文库，不需要复制 API Key。论文完成解析和翻译后，可以在手机或平板上下载为离线版本；电脑关机后，已下载论文、图片和本机阅读进度仍然可用。

## 第一次连接

1. 在电脑上启动 cark，但不要把端口直接暴露到公网：

   ```powershell
   cark gui --no-browser
   ```

2. 让电脑和手机加入同一个 Tailscale 网络，然后在电脑上运行：

   ```powershell
   tailscale serve --bg 8765
   ```

3. 用手机打开 Tailscale 给出的 HTTPS 地址。
4. iPhone/iPad 在浏览器分享菜单中选择“添加到主屏幕”；Android 在浏览器菜单中选择“安装应用”。

## 出门前

1. 打开要阅读的论文。
2. 点击底部工具栏的“离线”。
3. 等按钮变成“已离线”再离开网络。

离线状态下可以阅读原文与已生成的译文、查看图片、切换目录和保存本机阅读位置。解析新论文、生成翻译、AI 共读和跨设备同步仍需连接到运行中的 cark。

## 更新离线论文

电脑上的论文内容更新后，再点击一次“已离线”即可刷新设备上的副本。书架中的“已离线”标记表示这篇论文已经保存在当前设备。

> 不要直接使用路由器端口转发或把 `8765` 端口开放到公网。当前移动入口定位为个人设备使用，推荐只通过 Tailscale 访问。
