# cark 随身论文阅读器

移动端不需要连接电脑，也不需要复制 API Key。论文完成解析和翻译后，在电脑书架点击“导出手机文献包”，把生成的 `.carkpaper` 文件通过微信、网盘、数据线或系统分享发送到手机，App 直接导入即可阅读。

## 推荐：离线文献包

1. 在电脑上打开 cark 书架。
2. 点击论文卡片右侧的包裹图标“导出手机文献包”。
3. 把生成的 `.carkpaper` 文件发送到手机或平板。
4. 打开 cark App，点击“从手机选择 .carkpaper”并选择文件。
5. 校验通过后论文会直接出现在手机书架中；电脑关机、手机断网都能继续阅读。

文献包包含结构化原文、已有双语稿、目录、图片、批注和阅读位置。App 会校验包格式、论文身份和每个资源的 SHA-256；损坏或不完整的包不会写入书架。再次导入同一篇论文会更新设备上的副本。

## 可选：连接电脑同步

只有需要即时同步新内容时才需要以下步骤：

1. 在电脑上启动 cark，但不要把端口直接暴露到公网：

   ```powershell
   cark gui --no-browser
   ```

2. 让电脑和手机加入同一个 Tailscale 网络，然后在电脑上运行：

   ```powershell
   tailscale serve --bg 8765
   ```

3. 用手机打开 Tailscale 给出的 HTTPS 地址。
4. 在 App 中点击“连接电脑（可选）”，输入上一步的 HTTPS 地址。网页版本仍可从浏览器添加到主屏幕。

## 编译 Android APK

安装 Android Studio 及 Android SDK 后运行：

```powershell
cd gui
npm run android:apk
```

调试包输出到 `gui/android/app/build/outputs/apk/debug/app-debug.apk`。原生应用仅允许连接 HTTPS 地址，电脑端只对 Capacitor 本地来源开放跨域访问。
脚本会自动使用 Android Studio 自带的 JDK，并从常见位置寻找 Android SDK，无需手动设置环境变量。

## 旧的联网下载方式

1. 打开要阅读的论文。
2. 点击底部工具栏的“离线”。
3. 等按钮变成“已离线”再离开网络。

离线状态下可以阅读原文与已生成的译文、查看图片、切换目录和保存本机阅读位置。解析新论文、生成翻译、AI 共读和跨设备同步仍需连接到运行中的 cark；这些不影响已导入文献的普通阅读。

## 更新离线论文

电脑上的论文内容更新后，再点击一次“已离线”即可刷新设备上的副本。书架中的“已离线”标记表示这篇论文已经保存在当前设备。

> 不要直接使用路由器端口转发或把 `8765` 端口开放到公网。当前移动入口定位为个人设备使用，推荐只通过 Tailscale 访问。
